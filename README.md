# 🎪 豊橋技術科学大学　技科大祭実行委員会

ようこそ！私たちは豊橋技術科学大学の大学祭「技科大祭」を企画・運営している学生実行委員会です。

## 🌟 2026年 第49回技科大祭 ＆ 大学開学50周年！
2026年は、豊橋技術科学大学の開学50周年という記念すべき節目の年です。
私たち実行委員会は、例年以上に盛大で特別な「第49回技科大祭」を作り上げるため、局員一丸となって準備を進めています！
